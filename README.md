# InterviewIQ

An AI-powered mock interview platform that conducts realistic interview simulations, parses resumes, analyzes responses in real-time, and provides detailed feedback.

## Features
- **Resume Intelligence**: Parses experience, skills, and projects, generating an ATS compatibility score.
- **AI Interview Engine**: Google/Amazon/Microsoft/Startup/Finance style technical and behavioral interviews.
- **RAG Pipeline**: Semantic search to tailors questions to the candidate's actual projects.
- **Real-time Analytics**: Live feedback on confidence, communication, and technical depth.

## Stack
- **Frontend**: React, TypeScript, TailwindCSS, Zustand, React Query, Framer Motion
- **Backend**: Node.js, Express, PostgreSQL, Prisma ORM, Redis, Pinecone
