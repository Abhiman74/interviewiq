## 2. User Journeys
- **Persona: Student Placement Prep**:
  1. Signs up, selects "Student", fills branch/graduation year.
  2. Uploads resume, gets resume review (missing keywords like "DBMS", "REST").
  3. Sets up a "Beginner" "Software Engineer" interview with "Startup" style.
  4. Conducts interview, gets asked about projects, database queries.
  5. Reviews results, obtains custom practice link.
